# 商业级起名系统需求文档

## 一、系统目标

开发一个可商业化运营的“智能起名系统”，支持多类型起名，包括：

1. 个人姓名（宝宝起名、成人改名）
2. 公司/品牌名称
3. 产品/游戏/英文名等

系统特点：

* 智能生成（算法 + AI）
* 名字评分（音韵、字义、风格）
* 风格可选（现代、国风、科技、奢华等）
* 多渠道输出（小程序、Web站、API）
* 可商业化变现（会员、付费功能）
* SEO支持（内容生成，长尾覆盖）

---

## 二、系统架构

### 1. 用户层

* 小程序端
* H5/移动端网页
* Web后台管理
* 外部API调用（可选）

### 2. 核心服务层

* **词库系统**：管理汉字、组合词、行业词、风格标签、负面词
* **起名生成引擎**：根据规则、风格和输入生成名字候选
* **评分系统**：根据字义、音律、风格和名字结构打分
* **AI增强层**：提供名字解释、品牌故事、Slogan等
* **缓存层**：Redis/内存缓存，提升查询速度

### 3. 数据存储层

* SQLite（小规模）或 PostgreSQL（商业级）
* JSON/CSV（导入或更新词库）
* Redis（热门名字、评分缓存）

---

## 三、数据来源与词库设计

### 1. 核心数据来源

| 数据类型   | 来源/说明                       |
| ------ | --------------------------- |
| 汉字库    | Unicode CJK汉字标准、中文常用字表      |
| 拼音     | CEDICT、pinyin库              |
| 字义     | CEDICT、新华字典、公开字典            |
| 名字统计   | Chinese Names Corpus（性别、频率） |
| 行业/品牌词 | 规则生成或收集行业词                  |
| 负面/禁用词 | 字义消极、多音字歧义、不吉利词等            |

### 2. 词库分层

1. **核心字库**（5,000~8,000字）：高频字，适合名字生成
2. **扩展字库**（20,000~50,000字）：文学、古汉语、现代词汇
3. **行业/风格字库**（10,000~30,000字）：科技、AI、国风、奢华等
4. **负面过滤词**（5,000~10,000字）：不宜使用的字

---

## 四、名字生成逻辑

### 1. 输入参数

* 类型：个人/公司/品牌/产品
* 姓氏（可选）
* 行业/风格（可选）
* 名字长度（可选）
* 性别（个人姓名可选）
* 是否限制特定字库（可选）

### 2. 生成流程

1. **过滤字库**：去除负面词、用户不允许字
2. **选字规则**：

   * **个人姓名**：首字根据姓氏音律，次字根据寓意/风格
   * **公司/品牌**：行业词 + 风格词 + 组合后缀（科技/集团/系统）
3. **组合生成**：随机、权重组合或规则组合
4. **多轮候选**：生成 N 个候选名字

### 3. AI增强（可选）

* 给每个名字生成详细解释、品牌故事、Slogan
* 优化字义或音律
* 提供高端替代名

---

## 五、评分系统

### 1. 评分维度

| 维度      | 描述                | 权重  |
| ------- | ----------------- | --- |
| 字义正向性   | 字义是否积极吉利          | 20% |
| 音律顺滑度   | 名字音节搭配是否顺口、韵律是否和谐 | 20% |
| 风格匹配度   | 名字是否符合用户选择的风格/行业  | 15% |
| 常用度/可读性 | 字是否常用、容易书写和记忆     | 15% |
| 长度结构    | 名字长度是否符合个人或行业习惯   | 10% |
| 重复惩罚    | 避免名字中重复字          | 10% |
| AI综合评分  | 综合品牌故事、创意程度       | 10% |

### 2. 输出

* 综合得分（0~100）
* 可选排序：按评分高低或风格优先

## 六、系统功能模块

### 1. 小程序功能

* 首页输入姓名/类型/风格
* 名字生成候选列表
* 评分显示
* AI名字解释和品牌故事
* 收藏/分享
* 会员解锁更多生成数量

### 2. Web站功能（SEO）

* 长尾关键词覆盖：宝宝名字/公司名字/品牌名字/英文名字
* 页面自动生成：名字列表 + 解释 + Slogan
* 内容分享/收藏

### 3. API功能

* 接收请求参数：类型/长度/风格/姓氏
* 返回候选名字 + 评分 + AI解释
* 支持批量生成

### 4. 后台管理

* 词库维护：增删改查
* 风格/行业标签管理
* AI输出模板管理
* 会员/付费功能管理
* 数据统计（生成次数、热门名字）

---

## 七、商业化设计

1. **免费层**：每天生成数量有限，基础评分和解释
2. **付费会员**：解锁更多生成、AI品牌故事、Slogan、Logo提示词
3. **广告变现**：列表广告或激励视频
4. **SEO导流**：通过名字搜索、生成工具、长尾关键词
5. **数据分析**：热门名字/行业词统计，为内容和营销优化提供依据

---

## 八、技术实现逻辑总结

1. **数据获取**：集成CEDICT/Unicode/姓名语料/行业词，清洗合并到统一词库

2. **名字生成**：

   * 规则生成（姓+选字组合）
   * 权重选择（常用度、风格匹配、行业相关）
   * AI增强（品牌故事/解释/高端替代）

3. **评分系统**：多维度评分 + 权重加权，生成综合分数

4. **缓存优化**：Redis缓存热门名字和评分结果，提高性能

5. **输出**：Web、小程序、API、后台管理端

6. **付费与会员控制**：根据生成次数和功能限制，实现商业化

---
